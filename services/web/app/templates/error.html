  1. Visit the Issues page on our GitHub.
  2. Search the issue list to ensure that your report hasn't been filed already. If your report has already been filed by someone else, comment that you are experiencing the same problem - so we know that you're also affected.
  3. If you believe your bug is a new one, then feel free to create a "New Issue". Type in a complete summary and description of your issue and submit it.
